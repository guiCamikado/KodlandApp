# Sobre

Esta aplicação simples foi feita como pedido do processo seletivo da Kodland. Nela foi desenvolvido um gerenciador de candidaturas a vagas de empregos.

Ao se candidatar a uma ou mais vagas de emprego, algumas informações acabam se perdendo com o tempo e com a quantidade, como por exemplo: qual o nome da empresa em que se candidatou, quando se candidatou, o que a vaga pedia, se passou em parte do processo seletivo, se já recebeu uma mensagem sendo reprovado, etc.

O que esta aplicação busca é organizar das seguintes maneiras:

Uso de uma interface para registrar novas candidaturas

Essa interface conterá nome da empresa, link onde se cadastrou e data de cadastro.

Página de gerenciamento de candidaturas existentes

Candidaturas que já foram realizadas.

A ideia é utilizar banco de dados SQLite com interface Tkinter.

## Como executar a aplicação

Apenas executar o arquivo main.py com o console no local da pasta
